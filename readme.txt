Adds a WhatsApp button to your website that connects to a specified phone number.

== Description ==
The WhatsApp Button plugin adds a floating button to your website that, when clicked, opens a chat in WhatsApp with a specified phone number. You can customize the phone number from the WordPress dashboard. This plugin can be very useful for business websites that want to provide an easy way for customers to contact them directly through WhatsApp.

== Installation ==

Upload the plugin files to the /wp-content/plugins/whatsapp-button directory, or install the plugin through the WordPress plugins screen directly.
Activate the plugin through the 'Plugins' screen in WordPress.
Go to the WordPress dashboard -> Settings -> WhatsApp Button and enter your WhatsApp number.
The WhatsApp button will now appear on your website.
== Frequently Asked Questions ==
= How do I customize the phone number? =
You can change the phone number by going to the WordPress dashboard -> Settings -> WhatsApp Button and entering your desired phone number.

= How do I change the appearance of the WhatsApp button? =
You can customize the appearance of the WhatsApp button by modifying the CSS file located in the plugin directory.
